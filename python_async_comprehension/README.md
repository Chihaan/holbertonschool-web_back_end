# Python - Async comprehension

## Description
Utilisation des générateurs asynchrones et des compréhensions asynchrones en Python.

## Fichiers

| Fichier | Description |
|---|---|
| `0-async_generator.py` | Générateur asynchrone produisant des nombres aléatoires |
| `1-async_comprehension.py` | Compréhension de liste asynchrone à partir du générateur |
| `2-measure_runtime.py` | Mesure le temps d'exécution de compréhensions asynchrones concurrentes |
| `0-main.py` à `2-main.py` | Fichiers de test associés |
