# Python - Async function

## Description
Introduction à la programmation asynchrone en Python avec `asyncio` : coroutines, exécution concurrente, mesure de temps d'exécution et gestion de tâches.

## Fichiers

| Fichier | Description |
|---|---|
| `0-basic_async_syntax.py` | Coroutine basique avec un délai aléatoire |
| `1-concurrent_coroutines.py` | Exécution de plusieurs coroutines en parallèle |
| `2-measure_runtime.py` | Mesure le temps d'exécution total des coroutines |
| `3-tasks.py` | Création d'une tâche asyncio |
| `4-tasks.py` | Exécution concurrente de tâches asyncio |
| `0-main.py` à `4-main.py` | Fichiers de test associés |
